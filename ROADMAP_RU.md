# Roadmap Solink Workspace

## v0.1 - Файловая память и порядок

Цель: перестать терять контекст между чатами и проектами.

- Создать Manifest.
- Создать Projects Hub.
- Создать шаблон карточки проекта.
- Создать карточки ключевых проектов.
- Создать журнал решений.

## v0.2 - Первый Command Center

Цель: получить простой HTML-пульт для обзора проектов.

- Показать список проектов.
- Показать статус и momentum.
- Показать следующий шаг.
- Открывать нужные папки и документы через ссылки.
- Пока без запуска EXE.

## v0.3 - Command Service

Цель: добавить локальный сервис, который принимает команды от интерфейса.

- Реализовать Python-сервис.
- Добавить команды чтения статусов.
- Добавить безопасные команды открытия папок/файлов.
- Добавить журнал команд.

## v0.4 - Интеграция с Codex и Git

Цель: сделать Codex частью управляемого процесса.

- Карточки проектов должны указывать, какой контекст читать.
- Добавить команды подготовки задачи для Codex.
- Добавить Git-статус проектов.
- Добавить простую автоматизацию commit/report.

## v0.5 - Голос и смартфон

Цель: начать превращать систему в пульт управления с телефона.

- Большая кнопка микрофона.
- Простые голосовые команды.
- Отправка команд в Command Service.
- Быстрый статус проектов на мобильном экране.
