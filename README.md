# Solink Workspace

Персональная инженерная AI-операционная система для управления проектами, агентами, задачами и рабочим контекстом.

Это не Hermes, не Fugu, не ChatGPT и не Codex отдельно. Это собственная рабочая среда, в которую можно подключать разные модели, локальные сервисы и инструменты.

## Главные файлы

- [MANIFEST_RU.md](MANIFEST_RU.md) - правила системы, структура, роли агентов, принципы хранения контекста.
- [PROJECTS_HUB_RU.md](PROJECTS_HUB_RU.md) - главный список проектов и текущих следующих шагов.
- [DECISIONS_LOG_RU.md](DECISIONS_LOG_RU.md) - журнал важных архитектурных решений.
- [ROADMAP_RU.md](ROADMAP_RU.md) - порядок разработки системы.
- [templates/PROJECT_CARD_TEMPLATE_RU.md](templates/PROJECT_CARD_TEMPLATE_RU.md) - шаблон карточки проекта.

## Принцип работы

Чат не является единственным местом памяти. Чаты используются как рабочие комнаты, а долговременный контекст хранится в Markdown-файлах.

Любой новый агент сначала читает `MANIFEST_RU.md`, затем `PROJECTS_HUB_RU.md`, затем карточку конкретного проекта.
